# Blockchain_lab2
Используемые методы библиотеки ecdsa
Кривая SECP256k1:

    SECP256k1 - параметры эллиптической кривой Bitcoin
    .generator - базовая точка G генератор кривой
    .order - порядок n циклической подгруппы

Операции с точками:

    * - умножение точки на скаляр (k * G)
    + - сложение точек на кривой (R + e*P)
    .x() - получение x-координаты точки
    .y() - получение y-координаты точки

Математическая основа
Генерация ключей:

    Приватный ключ d = случайное число ∈ [1, n-1]
    Публичный ключ P = d * G

Создание подписи:

    Одноразовый номер k = случайное число ∈ [1, n-1]
    Точка R = k * G
    Хеш e = SHA256(R || P || сообщение)
    Подпись s = (k + e * d) mod n
    Результат: подпись = (R, s)

Проверка подписи:

    Хеш e = SHA256(R || P || сообщение)
    Левая часть: s * G
    Правая часть: R + e * P
    Проверка: s * G == R + e * P
